Firmware SupportÔÉÅ

X2Cscope firmware side library is written in standard C. Therefore the whole Microchip 16bit, 32bit MCU portfolio is supported.

https://x2cscope.github.io/docs/firmware/X2CscopeFirmware.html.

X2Cscope implementation is using dynamic function pointers. Some of our low-end 8bit MCUs does not support it. Contact us for more details: https://www.microchip.com/support

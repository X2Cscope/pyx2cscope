pyx2cscope.variable

variable package!

This package contains modules and classes for handling different types of variables stored in the MCU (Microcontroller Unit) data memory. It includes support for various data types and provides methods for reading, writing, and manipulating these variables using the LNet protocol.

Modules:
variable: Defines the base Variable class and specific variable types like

VariableInt8, VariableUint8, VariableInt16, etc.

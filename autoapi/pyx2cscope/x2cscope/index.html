pyx2cscope.x2cscope

X2CScope module for interfacing with the X2C firmware enabled embedded systems.

The pyx2cscope Python package communicates with X2Cscope enabled firmwares running on Microchip microcontrollers. It is utilising LNET protocol to communicate with the firmware. LNET protocol is implemented in the mchplnet package. The package provides an interface to connect to the firmware, set up scope channels, request data, and a process received data.

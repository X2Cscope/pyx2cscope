Bases: pyx2cscope.parser.Elf_Parser.ElfParser

Class for parsing ELF files generated by Microchip’s XC16 compiler.

This class extends the basic ElfParser to specifically handle ELF files produced by the XC16 compiler, extracting useful information about variables defined in the compiled program.

Attributes:

xc16_read_elf_path (str): Path to the XC16 readelf executable.

pyx2cscope.parser.elf16_parser

ELF16 Parser for Microchip’s XC16 Compiler.

This module defines the Elf16Parser class for parsing ELF files generated by Microchip’s XC16 compiler. The parser extracts useful information about variables defined in the compiled program.

Usage:

elf_reader = Elf16Parser(elf_file) variable_map = elf_reader.map_variables() print(variable_map)
